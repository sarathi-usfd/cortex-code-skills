**Author:** karthick

## Overview

Create, debug, optimize, and audit Snowflake Semantic Views for use with Cortex Analyst. Manages the semantic layer that enables natural language querying of Snowflake data, including logical/physical mappings, verified query recommendations (VQRs), and schema validation.

---

## When to Use

- Creating a new semantic view to enable natural language querying
- Debugging why Cortex Analyst generates incorrect SQL from a question
- Optimizing a semantic view for better query accuracy
- Auditing an existing semantic view for correctness and completeness
- Generating verified query recommendations (VQRs) to improve accuracy
- Validating semantic model schema before deploying to agents

---

## How to Use

Invoke for any semantic view task:

```
/semantic-view
```

**Or describe your intent:**
> "Create a semantic view for our sales data"
> "Why does Cortex Analyst give wrong results for revenue questions?"
> "Add verified query recommendations to the finance semantic view"
> "Validate the semantic view schema"

### Sub-skills

| Sub-skill | Purpose |
|-----------|---------|
| `creation` | Build a new semantic view from scratch or from tables |
| `debug` | Debug incorrect SQL or bad Cortex Analyst answers |
| `optimization` | Improve accuracy and query generation quality |
| `audit` | Verify configuration completeness and correctness |
| `vqr_suggestions` | Generate verified query recommendations |
| `validation` | Schema and syntax validation |
| `setup` | Initial environment setup |

### Key Scripts

| Script | Purpose |
|--------|---------|
| `scripts/semantic_view_get.py` | Export current semantic model to JSON |
| `scripts/semantic_view_set.py` | Apply semantic model changes |

---

## Testing

- Use `SHOW SEMANTIC VIEWS` to confirm creation
- Test natural language queries via Cortex Analyst after creating the view
- Use the `debug` sub-skill to trace incorrect SQL generation step-by-step
- Validate VQRs by running them and verifying results match expected output
- Run `validation` sub-skill before deploying to production agents
